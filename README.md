# Dynamic Pathfinding Agent

A real-time pathfinding visualization tool built with Python and Tkinter. The agent navigates a grid-based environment using informed search algorithms and can re-plan its path dynamically when obstacles appear.

## Features
- **Algorithms:** A* Search and Greedy Best-First Search.
- **Heuristics:** Manhattan and Euclidean distances.
- **Dynamic Mode:** Obstacles spawn randomly while the agent is moving.
- **Interactive UI:** Manually place walls, change grid size, and randomize maps.
